Blockly > Demos > Blind

Demo Goals

  • Demonstrate one way of displaying the traditional blockly code blocks in an accessible way for visually impaired users.
  • Demonstrate the ability to modify existing primitives (numbers, strings, etc).

Non-Goals

  • This demo does not validate or run the code displayed.
  • This demo does not allow the creation of new blocks or variables.
  • This demo does not support list or variable controls yet.

Instructions:

Click one of the block buttons to display the proposed output for that block.
